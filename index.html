¿Qué es una API?

Una API (Interfaz de Programación de Aplicaciones) permite que diferentes aplicaciones se comuniquen entre sí. Por ejemplo, cuando usas una app para ver el clima, esta usa una API para conectarse con un servidor que tiene los datos.
